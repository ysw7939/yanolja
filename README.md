# 도메인

## 상품 도메인

### 숙박업소 - accomodation

- 이름
- 위치
- 업소 유형 (몇성급 호텔, 모텔, 팬션 …)
- 전화번호
- 안내
- 정책 (프론트 운영시간)
- 언어 지원
- 판매자 정보
- 위도
- 경도

**테마 (주차가능, OTT, 와이파이…) N:N 관계**

**사진**

### 객실 - Room

숙박업소와 N:1 관계

- 기본정보
- 예약공지
- 체크인 시간
- 체크아웃 시간
- 가격
- 대실 - 이용시간
- 대실 가격

**객실 특성 (금연객실 , 기준인원/최대인원 , 더블침대, 싱글침대) N:N 관계**

**사진**

### 후기 - Review

- 작성자 이름
- 별점
- 작성 날짜
- 내용
- 판매자 답변
- 판매자 답변 날짜

**사진**
